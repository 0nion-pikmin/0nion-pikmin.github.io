1. What is the main characteristic of XQuery?

XQuery is a powerful, W3C-Recommended query language for extract, transform, and generate XML data. To navigate through XML documents, this language uses XPath expressions. Its easy to read and concise syntax means it's simpler to work with and easier to maintain, making XQuery an ideal replacement of complex Java or C++ programs

Compared to XSLT, XQuery is easier to learn, use, and maintain due to its similarity to database query language (e.g. SQL) and its less rigourous syntax rules. XQuery will neither run in Hills (college server) nor locally in your computer because it needs the XQuery machine

Use cases:

  • Extract information to use in a Web Service
  • Generate summary reports
  • Transform XML data to XHTML
  • Search Web documents for relevant information

2. Look at the XML in hw8.xml file. Look at the code in hw8.xq and based on your research, what is this code doing if applied on the XML of hw8.xml? You will find these files in the zipped folder you downloaded for this course)

hw8.xml's content

hw8.xq:

            
                xquery version "1.0";

                for $b in
                   doc("hw8.xml")/bookstore/book
                   
                where 
                   $b/price < 30
                
                return $b/title

Summary:

The provided XQuery code goes through each book entry in hw8.xml, look at each book's price, and then compare said price to see if it's less than 30. If that condition is true, that book entry will be printed out

Detailed explanation:

          
            for $b in
              doc("hw8.xml")/bookstore/book

for $b in: For each book element in

doc("hw8.xml"): Open hw8.xml file

/bookstore/book: XPath that XQuery uses to navigate through hw8.xml file by selecting bookstore element, then all book elements under bookstore

          
            where 
              $b/price < 30

            return $b/title

The XQuery code above will extract the price of each book entry in hw8.xml:

          35.00
        
          25.00
        
          48.99
        
          42.00
        
          5.89

If any of the extracted book's price is less than 30, the book titles that fit the requirement in

          
              $b/price < 30

will be printed out

Conclusion:

As the example above (hw8.xq) shows, as long as you know basic coding, you can read XQuery. This is powerful, versable, and beginner-friendly language

3. Do a brief research and explain what is XQIB and how it can be used with XQuery?

XQIB (XQuery-in-the-Browser) is HTML's cousin. Similar to HTML, It can program client-side Web apps, except with XQuery. This is a W3C Standard and works on most modern browsers (Internet Explorer, Firefox, Chrome, Safari). XQIB also doesn't require any plugin installation to run XQuery
